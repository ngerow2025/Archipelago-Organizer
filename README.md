# Archipelago Organiser

This project is a Qt-based organizer/launcher for the Archipelago modded games, multiworlds, and configuration files. The source code lives in `src/`, while the rest of the workspace is mostly generated by builds, overlay mounts, or local game/runtime setup.

## Workspace Layout

| Folder | Purpose |
| --- | --- |
| `Game/` | Local game install, force the use of windows compatibility tools in Steam then copy the contents of A Dance of Fire and Ice from there into this folder, `A Dance of Fire and Ice.exe` must end up in this folder. |
| `mod/ADOFAI_AP-Mod/` | ADOFAI Archipelago mod, download from https://github.com/ClaudeChibout/ADOFAI_AP-Mod/releases, `winhttp.dll` must be in this folder. |
| `BepInEx_win_x64_5.4.23.5` | BepInEx mod loader, download from https://github.com/BepInEx/BepInEx/releases/tag/v5.4.23.5, extract the contents of the zip into this folder. |
| `src/` | The source code for the application, including UI and database code. |
| `build/` | The build output directory, generated by the build process. |

## Notes
- The game, mod, and BepInEx must be downloaded separately, right now the paths are hardcoded
- all other folders generated are used during runtime for now
